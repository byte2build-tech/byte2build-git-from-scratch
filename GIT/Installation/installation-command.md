# Install Git on Linux

Step 1: Check if GIT is already installed
 Git –version
If the version is 2.20 or higher, can continue or upgrade the latest version.

Step 2:
For Debian / Ubuntu based systems:
sudo apt update
sudo apt install git

For Red Hat / CentOS / Rocky Linux:
sudo yum install git
or
sudo dnf install git


# Install Git on Windows
Step1 : Download the installer from the official Git website:
https://git-scm.com

Step 2: Run the installer and follow the setup instructions.
Step 3: Git will be available through Git Bash or the Command Prompt.
